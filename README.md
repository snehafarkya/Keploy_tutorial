# A Beginner-friendly tutorial to record and test Go APIs
I have used and run one of Keploy's Go quickstarts called Echo+Postgres.
I've followed the existing documentation and successfully recorded test cases. In this tutorial-like documentation, I have explained everything, with installation, deployment, docker and everything. This is totally a developer friendly tutorial designed for beginners, who might or might not have used Keploy before.
